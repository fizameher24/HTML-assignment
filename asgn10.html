Introduction

HTML entities are used to display reserved characters in HTML.

Special Characters

  • & - Ampersand (used for '&')
  • < - Less than (used for '<')
  • > - Greater Than (used for '>')
  • © - Copyright Symbol
  • ® - Registered Trademark Symbol
  • ™ - Trademark symbol

HTML Comments

Comments in HTML are not displayed in the browser but can be used to leave notes for developers. comments start with <!---- and end with --->

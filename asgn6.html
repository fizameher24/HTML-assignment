    Semantic html helps to improve the readability and accessability of your web pages by using meaningful tags that describesthe content they contain.

    we have used semantic elements such as header ,nav, main,article,aside, and footer to structure the content of the paage.This approach makes the document more understandable for humans and search engines.

    sematic html tags:

  • header-Defines header section.
  • nav-Defines navigation links.
  • article-defines an independent piece of content.
  • Section-Defines a section within a document.
  • footer-Defines a footer section.
  • aside-Defines a content aside from the main content.
  • main-Defines the main content.
